# uakomp
# 🇺🇦 Ukrainsk leksjon 1
Dette prosjektet inneholder en SCORM-basert leksjon i ukrainsk, utviklet for bruk i LMS (Learning Management System).
## innhold
* `index.html` – hovedfil for leksjonen
* `imsmanifest.xml` – SCORM-manifest
* `scorm_api.js` – SCORM API-integrasjon
* `audio/` – lydfiler for uttale og støtte
## Formål
Leksjonen er laget for voksne deltakere (A2-nivå) og fokuserer på:
* grunnleggende ordforråd
* uttale med lydstøtte
* praktisk språkbruk i arbeidssituasjoner
## Hvordan bruke
### I LMS:
1. Last ned prosjektet som ZIP
2. Last opp ZIP-filen i LMS (SCORM 1.2 eller 2004)
3. Start leksjonen via LMS
### Lokalt (for testing):
Åpne `index.html` i nettleser
Merk: SCORM-funksjoner virker kun i LMS
## 📁 Struktur

```
├── scorm_api.js
└── audio/
```
